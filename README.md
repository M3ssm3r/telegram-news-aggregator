# Telegram News Aggregator

Простенький бот на Python, который я накидал за вечер, чтобы перестать мониторить десяток сайтов вручную. 
Он хватает новости из RSS-ленты и кидает их в мой Telegram-канал. Работает через cron.

## Что он умеет?

-   Парсит любую RSS-ленту (в примере Хабр, потому что это святое).
-   Забирает N последних постов.
-   Отправляет их одним сообщением в телегу.

## Как запустить?

1.  `pip install feedparser python-telegram-bot`
2.  Создать бота в Telegram через `@BotFather` и получить токен.
3.  Создать публичный канал и сделать бота админом.
4.  Вписать свои токен и имя канала в `main.py`.
5.  Запустить `python main.py`.
