# Telegram News Aggregator Bot

Простой бот для автоматизации сбора и публикации новостей.

## Функционал

-   Подключается к RSS-ленте новостного сайта (в данном примере - habr.com).
-   Получает последние 5 заголовков новостей.
-   Форматирует их в аккуратное сообщение.
-   Отправляет сообщение в указанный Telegram-канал.

## Стек технологий

-   Python 3
-   `requests`
-   `feedparser`
-   `python-telegram-bot`
